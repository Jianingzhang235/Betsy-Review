### Installing Dependencies

From within the root directory:

```sh
#### Install
npm install -g webpack
npm install
npm install db-seed --save-dev
npm install moment --savesh
#### Run
mongod;
mongo
npm run seed
npm start
npm run react-dev (optional)

Open your browser on localhost: 3005
```
